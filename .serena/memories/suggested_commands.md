# Suggested Commands

## Development
- `task run` - Run the application locally
- `task build` - Build the application
- `task build-and-run` - Build and run
- `task test` - Run tests (`go test -v ./...`)
- `task install` - Install/tidy dependencies (`go mod tidy`)
- `task generate` - Regenerate API code from OpenAPI specs

## System Utils (macOS/Darwin)
- `git` - Version control
- `ls`, `find`, `grep` - File operations
- `task` - Task runner (Taskfile)
- `mise` - Tool version management
